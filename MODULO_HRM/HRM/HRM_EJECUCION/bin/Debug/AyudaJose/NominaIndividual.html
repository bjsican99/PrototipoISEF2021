En este formulario el usuario podr agregar percepciones o deducciones, por empleado,
como, por ejemplo:
HORAS EXTRAS
DESCUENTOS JUDICIALES
Para agregar una percepcin o deduccin, se debe tener en cuenta los siguientes puntos:
Seleccionar una nomina a la cual se agregarn los registros.
Seleccionar al empleado al que se agregaran los datos.
Seleccionar la percepcin o deduccin.
Llenar el campo de cantidad en el caso de que lo requiera, este campo le servir
cuando tenga que hacer clculos como el total de las horas extras multiplicado por el
valor de cada una, el valor se colocara en el campo valor.
Llenar el campo valor, el cual contendr el valor de la percepcin.
Luego de que el usuario halla llenado o seleccionado todos los datos deber presionar el
botn de Agregar.
Luego de presionar el botn de agregar, se agregara el clculo y la percepcin o deduccin
del empleado.
Esto lo puede hacer con cualquier empleado, tambin puede quitar una fila si lo requiere.
Para guardar los datos, deber presionar el botn de Guardar.
